Tourism Ontology Project

Overview
This project involves building and analyzing a tourism ontology using RDF/Turtle format. The ontology is populated with data on attractions, activities, hotels, restaurants, cities, and countries. It includes tools to convert CSV data to TTL format and a Jupyter notebook for querying the ontology with SPARQL and visualizing results (e.g., bar graphs) to answer competency questions such as the number of attractions per city or types of attractions.
The project demonstrates a data science pipeline for ontology management, data conversion, querying, and visualization using Python libraries like rdflib, pandas, and matplotlib.

Files

ONTOLOGYFINALLL.ttl: The primary Turtle file containing the tourism ontology, populated with entities like cities, countries, attractions, activities, hotels, and restaurants. Used as the base for SPARQL queries in the notebook.

csv_to_ttl.py: Python script to convert CSV data into Turtle (TTL) format for ontology population. It processes an example CSV file and generates RDF triples.

queries-result(1).csv: Sample CSV input file used by csv_to_ttl.py to demonstrate data conversion (e.g., containing tourism-related data like locations and categories).

test_output.ttl: The output Turtle file generated by running csv_to_ttl.py on the example CSV. This can be merged into the main ontology or used for testing.

queries.ipynb: Jupyter notebook that loads the main ontology (ONTOLOGYFINALLL.ttl), runs SPARQL queries to extract data, creates Pandas DataFrames, and generates bar graph visualizations to answer competency questions (e.g., attractions per city, activities per city, attraction types).

Setup and Dependencies

Python 3.x
Required libraries: rdflib, pandas, matplotlib (install via pip install rdflib pandas matplotlib)

Usage

Convert CSV to TTL:

Run python csv_to_ttl.py 
Output: test_output.ttl (generated RDF triples).


Query and Visualize:

Open queries.ipynb in Jupyter Notebook or JupyterLab.
Ensure ONTOLOGYFINALLL.ttl is available.
Run cells to parse the ontology, execute SPARQL queries, and display bar graphs for competency questions.



Competency Questions Addressed
The notebook visualizes answers to questions like:

Number of touristic attractions in different cities.
Specific things to do (activities) in a city.
Types of attractions (e.g., cultural, historic) per city.
